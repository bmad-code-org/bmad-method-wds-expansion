# Step 02: Generate Visual Representation

**Goal:** Create the visual design using the chosen tool.

---

## Process

### By Tool

**Nano Banana:**
1. Compose prompt describing the page layout, sections, and style
2. Generate image
3. Present to user for feedback
4. Iterate if needed

**Figma:**
1. Guide user through creating the design in Figma
2. Or interpret a Figma export/screenshot
3. Document design decisions

**HTML Prototype:**
1. Generate HTML/CSS for the page layout
2. Include key components and content
3. Present for review

**Wireframe:**
1. Create ASCII or simple wireframe description
2. Focus on layout and component placement
3. Present for review
